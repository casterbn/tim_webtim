## webtim 使用timjs 实现的web通讯工具

###### webtim主要展示 调用tim的部分接口，实现的通讯功能：注册，登录，加好友，建群，申请入群，退群，踢出群，上线，离线，退出 等接口，包括部分后台接口，部分接口未调用，如，拉黑，黑名单，撤回消息，阅后即焚，流数据，推流等等，若需用了解，请查看tim文档 https://tlnet.top/timdoc  或等待webtim的功能完善
